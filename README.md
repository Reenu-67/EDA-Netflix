# 🎬 Netflix Exploratory Data Analysis (EDA)

## 📌 Project Overview

This project performs **Exploratory Data Analysis (EDA)** on the Netflix Movies and TV Shows dataset using Python. The objective is to understand Netflix's content library by analyzing content types, genres, countries, ratings, release trends, duration, and other important characteristics through statistical analysis and visualizations.

The project demonstrates the complete data analysis workflow, including data cleaning, feature engineering, visualization, outlier detection, and business insight generation.

---

## 🎯 Objectives

* Understand the structure of the Netflix dataset.
* Clean and preprocess the data.
* Perform univariate, bivariate, and multivariate analysis.
* Identify trends in content production over time.
* Analyze genres, ratings, countries, and duration.
* Detect outliers using statistical methods.
* Create new features through feature engineering.
* Generate meaningful business insights and recommendations.

# 📈 Key Findings

* 🎬 Movies account for **69%** of Netflix's content.
* 🌎 United States is the largest content producer, followed by India.
* 🎭 International Movies are the most common genre.
* 📅 2018 was the peak content release year.
* 📺 TV-MA is the most common content rating.
* ⏱️ Average movie duration is **99.31 minutes**.
* 📚 Most content belongs to the **2010s** decade.
* 🎥 Most titles fall into the **Medium Duration** category.
